  • `pd.DataFrame(data)`: Membuat dataframe.
  • `df.head(n)`: Menampilkan n baris pertama.
  • `df.tail(n)`: Menampilkan n baris terakhir.
  • `df.info()`: Menampilkan Informasi Data Secara Ringas.
  • `df['column_name']`: Select a single column.
  • `df[['col1', 'col2']]`: Select multiple columns.
  • `df.loc[row_label, col_label]`: Access a group of rows and columns by labels.
  • `df.iloc[row_index, col_index]`: Access a group of rows and columns by integer position.
